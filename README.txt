SIMPLE PHP CRAWLER

This is a crawler used in my Search Engine Optimization class

It utilizes SIMPLE HTML DOM PARSER which can be found at:

http://simplehtmldom.sourceforge.net/manual.htm


Basics:
1. include Simple HTML DOM Parser
2. choose what you want to index in function indexPage($html);
3. execute function with url & depth [i.e. crawlPage('http://google.com/', 5)]
